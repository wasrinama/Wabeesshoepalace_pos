## 🛠️ **Troubleshooting**

### **Problem: "No printer detected"**
**Solutions:**
1. ✅ Check printer is **ON** and **connected**
2. ✅ Print test page from Windows/Mac settings
3. ✅ Try refreshing the browser page
4. ✅ Use **"System Driver"** option instead

### **Problem: "Print quality poor"**  
**Solutions:**
1. ✅ Check printer paper/ink levels
2. ✅ Update printer drivers
3. ✅ Use **"Direct"** printing method
4. ✅ Adjust print settings in Windows/Mac

### **Problem: "Barcode not scanning"**
**Solutions:**
1. ✅ Ensure barcode is **not too small**
2. ✅ Use **label printer** for best quality
3. ✅ Check barcode format is correct
4. ✅ Clean printer head

### **Problem: "Permission denied"**
**Solutions:**
1. ✅ **Allow printer access** in browser
2. ✅ Use **"System Driver"** method
3. ✅ Check Windows printer sharing settings
4. ✅ Run browser as administrator (if needed)
